# Portafolio de Proyectos – TripleTen
Este repositorio reúne los principales proyectos desarrollados durante el bootcamp de
Data Analyst en TripleTen. 

## Objetivo 
Demostrar habilidades en análisisde datos, visualización, desarrollo de dashboards interactivos y despliegue de aplicaciones
web con Python.

## Sobre mí
Soy Analista de Datos con formación en ingeniería y experiencia en el uso de Python para el
análisis exploratorio de datos, visualización y desarrollo de aplicaciones interactivas.
Me interesa transformar datos en información útil que apoye la toma de decisiones.

## Proyectos

### Dashboard de Anuncios de Autos
Proyecto desarrollado con Python, Streamlit y Plotly Express para la construcción y despliegue
de un panel de control interactivo orientado al análisis de anuncios de venta de autos.

**Funcionalidades:**
- Análisis exploratorio de datos (EDA).
- Histograma del kilometraje (odómetro).
- Gráfico de dispersión precio vs año del vehículo.
- Interfaz interactiva con Streamlit.

Repositorio:  
https://github.com/SofCast27/mi_proyecto_sprint7  

Demo:  
https://dashboard-autos-sofcast27.streamlit.app/

### Simulación de Lanzamiento de Moneda
Aplicación interactiva desarrollada con Python y Streamlit que simula el lanzamiento de una
moneda para analizar la probabilidad de ocurrencia.

El proyecto permite ejecutar múltiples simulaciones y visualizar cómo evolucionan las
probabilidades a medida que aumenta el número de intentos, mostrando su convergencia hacia
la probabilidad teórica.

Repositorio:  
https://github.com/SofCast27/streamlit_test  

Demo:  
https://test-sofcast27.streamlit.app/
