## "portafolio_tripleten"
Portafolio para proyectos desarrollados durante el curso en TripleTen.

# Proyectos incluidos
- `mi_proyecto_sprint7`
  Proyecto desarrollado con Python, Streamlit y Ploty Express para la construcción y despliegue de un panel de control interactivo orientado al análisis de anuncios de venta de autos.
  Enlace de la aplicación:
  https://dashboard-autos-sofcast27.streamlit.app/
  
- `streamlit_test`
  Aplicación interactiva en Python y Streamlit que simula el lanzamiento de una moneda. Proyecto desarrollado con Python, Pandas, Streamlit, Github y Scipy.
  Enlace de la aplicación:
  https://test-sofcast27.streamlit.app/
